試撰寫一程式，可由鍵盤輸入英哩，程式的輸出為公里，其轉換公式如下：<br>
1 英哩= 1.6 公里<br>

<h3>輸入說明</h3><br>
輸入欲轉換之英哩數(int)。<br>

<h3>輸出說明</h3><br>
輸出公里(double)，取到小數點以下第一位。<br>

<h3>輸入範例</h3><br>
90<br>
95<br><br>

<h3>輸出範例</h3><br>
144.0<br>
152.0
