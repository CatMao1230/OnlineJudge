輸入所使用的度數，換算夏月及非夏月之電費金額。<br>

|每度（元）|夏月|非夏月|
|---|---|---|
|120 度以下部分|2.10|2.10|
|121-330 度部分|3.02|2.68|
|331-500 度部分|4.39|3.61
|501-700 度部分|4.97|4.01|
|701 度以上部分|5.63|4.50|<br>

<h3>輸入說明</h3>
一個度數 ( 正整數 )。<br>

<h3>輸出說明</h3>
夏月與非夏月的金額，請輸出至小數點後兩位。<br>

<h3>輸入範例</h3>
525<br>

<h3>輸出範例</h3>
Summer months:1756.75<br>
Non-Summer months:1528.75<br>
