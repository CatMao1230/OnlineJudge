迴文是指從前面讀和從後面讀都相同的一個數字或一段文字。例如下列每一五位數的整數都是迴文： 123321 ， 55555 ， 45554 ， 11611 。請撰寫一個程式，判斷它是否迴文。<br>

<h3>輸入說明</h3>
輸入一個正整數。<br>

<h3>輸出說明</h3>
迴文印出 YES ；非回文印出 NO 。<br>

<h3>輸入範例</h3>
123321<br>
1556551<br>
1244221<br>

<h3>輸出範例</h3>
YES<br>
YES<br>
NO<br>
