OX 遊戲是大家小時候的童年記憶，甚至在長大後，無聊時還是會玩一下呢 ! 但當你一個人時，是否也能讓電腦跟你玩呢 ?<br>

<h3>輸入說明</h3>
輸入一個 3*3 的 2 維矩陣，矩陣內容， 0 代表 O ， 1 代表 X 。註：每列數字之間均有空格隔開。<br>

<h3>輸出說明</h3>
輸出這場 OX 遊戲是否有勝負之分。註：輸出結尾需換行。<br>

<h3>輸入範例</h3>
1 0 1<br>
0 0 1<br>
1 0 0<br><br>
1 0 0<br>
0 1 1<br>
0 1 0<br>

<h3>輸出範例</h3>
True<br>
False<br>
