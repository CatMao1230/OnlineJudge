試寫一個程式，輸入一正整數N，可計算出1到N之間可被3整除的數值之總和。<br>

<h3>輸入說明</h3>
輸入一正整數。<br>

<h3>輸出說明</h3>
輸出總和。<br>

<h3>輸入範例</h3>
100<br>
150<br>

<h3>輸出範例</h3>
1683<br>
3825<br>
