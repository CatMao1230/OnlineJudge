已知男生標準體重＝(身高－80 )*0.7；女生標準體重＝(身高－70)*0.6；試寫一個程式可以計算男生女生的標準體重。<br>

<h3>輸入說明</h3>
每個測資檔案會有多組測資案例。輸入兩個數值，依序代表為身高及性別（1代表男性；2代表女性）。<br>

<h3>輸出說明</h3>
輸出標準體重，浮點數取至第一位。<br>

<h3>輸入範例</h3>
172 1<br>
165 2<br>

<h3>輸出範例</h3>
64.4<br>
57.0<br>
