試撰寫一個程式，由輸入一個整數，然後判別此數是否為質數。質數是指除了 1 和它本身之外，沒有其它的數可以整除它的數，例如， 2, 3, 5, 7 與 11 等皆為質數。<br>

<h3>輸入說明</h3>
輸入一個正整數。<br>

<h3>輸出說明</h3>
質數顯示 YES ；非質數顯示 NO 。<br>

<h3>輸入範例</h3>
23<br>
37<br>
39<br>

<h3>輸出範例</h3>
YES<br>
YES<br>
NO<br>
