設計一個程式可以檢查身分證字號的正確性 ( 應檢查性別欄及檢查碼是否正確 ) 。<br>

<h3>輸入說明</h3>
輸入身分證字號，第一碼為英文大寫。<br>

<h3>輸出說明</h3>
若身分證字號正確，印出「CORRECT!!!」；不正確則印出「WRONG!!!」。<br>

<h3>輸入範例</h3>
A123456789<br>
L163690274<br>

<h3>輸出範例</h3>
CORRECT!!!<br>
WRONG!!!<br>
