給定下列遞迴函式，請計算出 f (k) ：<br>
f(n) = n + 1, when n = 0, n = 1<br>
f(n) = f(n - 1) + f(|n / 2|), when n > 1<br>

<h3>輸入說明</h3>
輸入值為一個大於 1 的整數。<br>

<h3>輸出說明</h3>
f(k) 的計算結果。<br>

<h3>輸入範例</h3>
10<br>
12<br>

<h3>輸出範例</h3>
60<br>
94<br>
