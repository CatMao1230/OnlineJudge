撰寫一個程式，使用者輸入一個整數，印出8位元的二進制表示。<br>

<h3>輸入說明</h3>
輸入一個整數，介於-128～127之間。<br>

<h3>輸出說明</h3>
以8位元的二進制顯示。<br>

<h3>輸入範例</h3>
15<br>
-2<br>

<h3>輸出範例</h3>
00001111<br>
11111110<br>
